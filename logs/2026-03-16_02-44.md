### ツールページのデザインをブログ側に統一

**対象ファイル:**
- `app/tools/memphis/page.tsx`
- `app/tools/trimming/page.tsx`

**変更内容:**

ブログのデザインシステム（`globals.css` で定義されたCSS変数）に合わせて、ハードコードされた色クラスをテーマ変数に置き換えた。

| 変更前 | 変更後 |
|--------|--------|
| `bg-white`, `bg-gray-50`, `bg-slate-50` | `bg-bg`, `bg-card`, `bg-secondary` |
| `border-slate-200`, `border-gray-200` | `border-border` |
| `text-slate-600`, `text-slate-700` | `text-text` |
| `text-slate-400`（サブテキスト） | `text-text/50`, `text-text/70` |
| `border-indigo-*`, `bg-indigo-*`（選択状態） | `border-accent`, `bg-accent/10`, `text-accent` |
| グラデーション・`bg-indigo-600` ボタン | `bg-accent`（emerald系） |
| `accent-indigo-600`（range slider） | `accent-accent` |
| `shadow-2xl shadow-indigo-100` | `shadow-lg` |

**効果:**
- ライト／ダークモードの自動対応（CSS変数が `.dark` クラスで切り替わるため）
- ブログ側と同じエメラルドグリーン（`--color-accent: #10b981`）のアクセントカラーに統一
