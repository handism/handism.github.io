# DESIGN_BAUHAUS.md — Bauhaus テーマ詳細仕様

## コンセプト
1919年創立のバウハウス（Bauhaus）スクールの美学を現代UIに再解釈。「形態は機能に従う」という原則のもと、赤・黄・青の三原色と黒を用いた幾何学的・機能的なデザイン。

## カラーパレット

### ライトモード
| 用途 | 値 | 説明 |
|------|-----|------|
| `--color-bg` | `#f5f5f0` | オフホワイト（未染色の紙） |
| `--color-card` | `#ffffff` | ピュアホワイト |
| `--color-text` | `#1a1a1a` | ほぼ黒 |
| `--color-border` | `#1a1a1a` | 太い黒ライン |
| `--color-accent` | `#e63329` | バウハウスレッド |

### ダークモード
| 用途 | 値 | 説明 |
|------|-----|------|
| `--color-bg` | `#0f0f0f` | 深い黒 |
| `--color-card` | `#1a1a1a` | ほぼ黒 |
| `--color-text` | `#f5f5f0` | オフホワイト |
| `--color-accent` | `#ff4438` | 明るいバウハウスレッド |

## フォント
- **見出し/本文**: `'Bebas Neue', 'Space Grotesk', sans-serif`
- 幾何学的・太い無機質なサンセリフ体（Neue Helveticaに近い）

## タイポグラフィ
- h1: `3.5rem / line-height: 1`（巨大で力強い）
- h2: `2.5rem / line-height: 1.1`
- すべての見出し: `letter-spacing: 0.1em`

## カードスタイル
- ボーダー幅: `2px`（太い黒ライン）
- **角丸なし**: `border-radius: 0px`（直角が幾何学的美学）
- シャドウなし
- ホバー: 境界線色が赤に変化（アクセントカラー）

## ボタンスタイル
- ボーダー幅: `3px`（さらに太い）
- 背景: 透明（ゴーストボタン）
- フォント: `Bebas Neue`
- ホバー: 黒で塗りつぶし（反転効果）
- 文字間隔: `0.12em`

## 三原色アクセント
- 赤（#e63329）: メインアクセント・h4ボーダー・カードホバーボーダー
- 黄（#f5c400）: blockquoteボーダー
- 青（#2563eb）: コードブロック左ボーダー
