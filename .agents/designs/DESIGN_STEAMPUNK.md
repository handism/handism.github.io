# DESIGN_STEAMPUNK.md — Steampunk テーマ詳細仕様

## コンセプト
19世紀ヴィクトリア朝の蒸気機関時代を想起させる重厚なデザイン。真鍮・革・木材・歯車のテクスチャーと濃いコーヒーブラウンの背景で、時代を超えた機械美学を表現する。

## カラーパレット

### ライトモード（深いブラウン基調）
| 用途 | 値 | 説明 |
|------|-----|------|
| `--color-bg` | `#2a1a0a` | 深いコーヒーブラウン |
| `--color-card` | `#3d2512` | 革のような茶色 |
| `--color-text` | `#e8c99a` | アンティーク羊皮紙色 |
| `--color-border` | `#8b6535` | 真鍮色 |
| `--color-accent` | `#c8872a` | 輝く真鍮ゴールド |

### ダークモード
| 用途 | 値 | 説明 |
|------|-----|------|
| `--color-bg` | `#100808` | 炭の黒に近いブラウン |
| `--color-accent` | `#e09840` | より明るい真鍮色 |

## フォント
- **見出し/本文**: `'Cinzel', 'Noto Serif JP', serif`
- ヴィクトリアン朝の格調あるセリフ体

## 背景
ラジアルグラデーション: 深いコーヒーブラウンから黒に向かう有機的な暗さ

## カードスタイル
- ボーダー幅: `2px`（真鍮フレーム感）
- 角丸: `6px`
- シャドウ: 内側光沢 + 外側の重厚な影
- ホバー: `translateY(-2px)` + 真鍮色のアクセント

## ボタンスタイル
- グラデーション背景（金属光沢）
- フォント: `Cinzel`
- 文字間隔: `0.08em`（格調ある間隔）
- アクティブ時: わずかに沈む押し込み感

## 特徴的なCSSパターン
```css
/* 見出しスタイル */
h1, h2, h3 {
  font-family: 'Cinzel', serif;
  letter-spacing: 0.06em;
  color: #d4aa6a; /* 真鍮ゴールド */
}

/* ボタングラデーション */
--theme-btn-bg: linear-gradient(180deg, #5a3a18 0%, #3d2512 100%);
```
