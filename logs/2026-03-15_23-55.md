### 1. コードブロックのファイル名表示（```lang:filename 記法）

**対象ファイル:**
- `src/lib/post-renderer.ts`
- `app/globals.css`

**変更内容:**

` ```ts:filename.ts ` のような記法でコードブロックにファイル名を表示できるようにした。

| コンポーネント | 処理 |
|--------------|------|
| `remarkExtractCodeFilename`（新規追加） | Remark プラグイン。`lang` に `:` が含まれる場合、`lang:filename` を分離して `lang="ts"` + `meta='filename="filename.ts"'` に変換 |
| Shiki トランスフォーマー | `meta.__raw` を解析して `data-filename` を `<pre>` に付与 |
| CSS | `pre.shiki[data-filename]::before` でファイル名をコードブロック左上に表示（モノスペースフォント・大文字化なし） |

**使い方:**
````markdown
```ts:utils/helper.ts
export function greet(name: string) {
  return `Hello, ${name}!`;
}
```
````

### 2. 記事の文字数・読了時間の精度向上

**対象ファイル:**
- `src/lib/post-parser.ts`
- `src/lib/utils.ts`
- `src/components/PostMeta.tsx`

**変更内容:**

`markdownToPlaintext` を強化し、`estimateReadingMinutes` を CJK 対応の計算に刷新した。

**`markdownToPlaintext` の改善:**

| 除去対象 | 旧 | 新 |
|---------|----|----|
| フェンスコードブロック ` ``` ` | 未対応（文字数に混入） | 除去 |
| 画像 `![]()` | 未対応 | 除去 |
| 引用マーカー `>` | 未対応 | 除去 |
| テーブル `\|` | 未対応 | 除去・区切り行削除 |
| HTML タグ | 未対応 | 除去 |
| 太字・斜体 | 記号ごと除去（テキスト消滅） | 記号のみ除去・テキスト保持 |

**`estimateReadingMinutes` の改善:**

```
読了時間（分）= ceil( CJK文字数 ÷ 600 + 英単語数 ÷ 200 )
```

旧実装の `文字数 ÷ 500`（日英区別なし）から、日本語(CJK)と英語を別速度で計算する方式に変更。
