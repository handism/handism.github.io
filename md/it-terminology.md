## IT用語集

開発や運用でよく出てくるIT用語を、カテゴリ別に短く整理しています。

### インフラ/ネットワーク

ネットワークや基盤を支える要素をまとめています。

| 用語                       | 説明                                                   |
| -------------------------- | ------------------------------------------------------ |
| `コロケーション`           | データセンターなどに自社機器を設置して運用すること。   |
| `パケットキャプチャー`     | ネットワーク上を流れるパケットを取得して解析すること。 |
| `トランジット`             | 他者のネットワークを経由して通信すること。             |
| `ピアリング`               | 異なるネットワーク同士を直接接続すること。             |
| `バックボーンネットワーク` | 主要ネットワークを結ぶ高容量のコア網。                 |
| `トポロジー`               | ネットワークやシステムの構成・接続形態。               |
| `流量制限`                 | 帯域や通信量に上限を設けること。                       |

### セキュリティ/認証

認証やデータ保護に関わる基本用語です。

| 用語                   | 説明                                                         |
| ---------------------- | ------------------------------------------------------------ |
| `ウォーターマーク`     | 画像や文書に埋め込む識別・著作権情報。                       |
| `ソルト`               | ハッシュ化時にランダム値を追加して安全性を高めること。       |
| `ペッパー`             | ハッシュ化時に秘密のランダム値を追加して安全性を高めること。 |
| `ストレッチング`       | ハッシュ計算を意図的に遅くして安全性を高めること。           |
| `JWT`                  | JSON Web Token。情報を安全に伝達するためのトークン。         |
| `セキュリティチェーン` | 対策を鎖に例え、弱点を埋める考え方。                         |

### 開発/プログラミング

開発プロセスや実装で頻出する言葉を収録しています。

| 用語         | 説明                                     |
| ------------ | ---------------------------------------- |
| `糖衣構文`   | 冗長な記述を簡潔にするための構文。       |
| `Webhook`    | イベント発生時に外部へ通知を送る仕組み。 |
| `シーケンス` | 順序付けられた処理やイベントの流れ。     |

### データ/キャッシュ

データ管理やキャッシュ運用で出てくる用語です。

| 用語             | 説明                                         |
| ---------------- | -------------------------------------------- |
| `サロゲートキー` | 主キーの代わりに使う代理キー。               |
| `エビクション`   | キャッシュやDBから不要データを削除すること。 |

### 信頼性/運用

安定運用や障害対応に関する用語をまとめています。

| 用語               | 説明                                           |
| ------------------ | ---------------------------------------------- |
| `外形監視`         | 外部からサービスの可用性や応答を監視すること。 |
| `フェイルオーバー` | 障害時に別系統へ自動切り替えすること。         |
| `冪等性`           | 同じ操作を繰り返しても結果が変わらない性質。   |

### テスト/品質

品質保証やテスト手法の基礎用語です。

| 用語               | 説明                                           |
| ------------------ | ---------------------------------------------- |
| `ファジングテスト` | ランダムデータで安全性・信頼性を検証する手法。 |
| `スモークテスト`   | 重大な問題がないかをざっくり確認するテスト。   |

### アルゴリズム

計算手法や最適化に関わる代表的な用語です。

| 用語             | 説明                                           |
| ---------------- | ---------------------------------------------- |
| `ダイクストラ法` | 重み付きグラフで最短経路を求めるアルゴリズム。 |

### ブロックチェーン

分散台帳やスマートコントラクトに関する用語です。

| 用語                   | 説明                                                 |
| ---------------------- | ---------------------------------------------------- |
| `スマートコントラクト` | 条件成立時に自動実行されるブロックチェーン上の契約。 |

### AI

AI分野で頻出する用語を簡潔にまとめています。

| 用語               | 説明                                       |
| ------------------ | ------------------------------------------ |
| `ハルシネーション` | 存在しない情報を生成・認識してしまうこと。 |

### コミュニケーション

チーム開発やレビューで使われる用語です。

| 用語   | 説明                                             |
| ------ | ------------------------------------------------ |
| `LGTM` | Looks Good To Me。レビューで承認を示すコメント。 |

### UX/人間要因

ユーザー体験や人間工学に関する用語です。

| 用語         | 説明                             |
| ------------ | -------------------------------- |
| `ポスチャー` | ユーザーの姿勢や動作などの態度。 |

### ドキュメンテーション

資料作成や説明に関わる用語をまとめています。

| 用語       | 説明                               |
| ---------- | ---------------------------------- |
| `ポンチ絵` | ざっくり概要を伝えるための簡易図。 |
