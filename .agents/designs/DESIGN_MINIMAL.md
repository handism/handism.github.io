## Overview
Minimal / Clean（ミニマル/クリーン）は、不要な装飾や境界を排除し、コンテンツそのものとタイポグラフィの美しさにフォーカスしたデザインスタイルです。
広い余白（ネガティブスペース）、細く繊細なボーダー、控えめでソフトなシャドウ、および洗練されたニュートラルカラーを基盤とします。
認知的負荷を極限まで減らし、ユーザーが情報に集中できるフラットでモダンな読書体験を提供します。

## Colors
白や淡いグレーなどのクリーンな無彩色をベースに、プライマリーアクセントとして知的なブルーを採用しています。

### ライトモード (標準)
- **プライマリー (`primary`):** 信頼感を与えるブルー (`#3b82f6`)。重要なリンクやアクセントに使用されます。
- **セカンダリー/テキスト (`secondary`):** 暗いスレートグレー (`#111827`)。漆黒ではなくややトーンを落とし、目に優しいコントラストを作ります。
- **ニュートラル/背景 (`neutral`):** 純白 (`#ffffff`)。余白の美しさを最大化するためのキャンバスです。
- **カード背景 (`card`):** ごく薄いグレー (`#f9fafb`)。背景と緩やかに境界を分けるために使用します。

### ダークモード
- **プライマリー (ダーク):** 明るいブルー (`#60a5fa`)。
- **セカンダリー (ダーク):** 明るいグレーホワイト (`#f9fafb`)。
- **ニュートラル (ダーク):** 暗いネイビーグレー (`#111827`)。
- **カード背景 (ダーク):** ソリッドなダークグレー (`#1f2937`)。

## Typography
洗練されたモダンな印象を際立たせるため、サンセリフフォントを使用します。
- **メインフォント:** `Lexend`。幾何学的で無駄のない字形が、ミニマルな空間設計に完全に調和します。

## Layout
このテーマにおいては「余白」が主役です。
コンテンツ同士を過度に近づけず、十分なマージン（`spacing.lg`: 24px やそれ以上）を確保します。
境界線は極力省略するか、極細の淡いボーダー（ライトモードでは `#e5e7eb`）を適用し、グリッドはすっきりと整列させます。

## Elevation & Depth
立体感は最小限に抑え、ほぼフラットに表現します。
- **カードシャドウ:** 非常に浅く柔らかい影を適用します。
  `0 1px 3px rgba(0, 0, 0, 0.06), 0 1px 2px rgba(0, 0, 0, 0.04)`。
- **ホバー時挙動:** 微小な浮遊感を与えるため、ホバー時に `translateY(-2px)` 移動させ、シャドウをわずかに深くします (`0 4px 12px rgba(0, 0, 0, 0.08)`)。
- **ボタン挙動:** シャドウは基本的に持たせずフラットとし、ホバー時に背景をセカンダリーカラーに変更するなどのカラーシフトで表現します。

## Shapes
過度な丸みを避け、現代的なシャープさを演出します。
- **カード角丸 (`rounded.card`):** `12px`。スマートで洗練された角丸です。
- **ボタン角丸 (`rounded.btn`):** `8px`。

## Components
### カード (`.theme-card`)
細いボーダー (`border: 1px solid var(--color-border)`) とごく薄い背景色、控えめな影の組み合わせ。

### ボタン (`.theme-btn`)
細いボーダー、滑らかなホバー背景色の遷移。余計な変形や3D表現は適用しません。

## Do's and Don'ts
### Do's
- 十分な余白（ネガティブスペース）を確保し、画面が要素で埋め尽くされないように配置してください。
- 影や境界線は「かろうじて視認できる」程度の淡さに設定し、階層をフラットに保ってください。

### Don'ts
- 太い黒枠線や、彩度の高すぎるハードシャドウは絶対に使用しないでください。
- 必要以上に多くのアクセントカラーを混在させないでください。メインアクセントのブルーとニュートラルグレーのみでシンプルにまとめます。
