# DESIGN_BLUEPRINT.md — Blueprint テーマ詳細仕様

## コンセプト
CAD/工学設計図・青写真（Blueprint）のような精密な技術的美学。濃紺のグリッド背景に細い境界線が走り、エンジニアの「設計書」感を演出する。

## カラーパレット

### ライトモード
| 用途 | 値 | 説明 |
|------|-----|------|
| `--color-bg` | `#e8f0ff` | 薄い青みがかったホワイト |
| `--color-card` | `#d0e0ff` | 淡いブループリント紙色 |
| `--color-text` | `#0a1628` | 濃紺（墨色に近い） |
| `--color-accent` | `#1a4fa0` | ロイヤルブルー |

### ダークモード
| 用途 | 値 | 説明 |
|------|-----|------|
| `--color-bg` | `#040d1e` | 夜の設計室のような深い濃紺 |
| `--color-card` | `#081530` | 濃紺カード |
| `--color-text` | `#c8deff` | 淡いブルーホワイト |
| `--color-accent` | `#60a5fa` | 明るいスカイブルー |

## フォント
- **見出し/本文**: `'Roboto Mono', 'Space Grotesk', monospace`
- 等幅フォントで技術・コード感を演出

## 背景
4層のグリッドパターン（大グリッド80px + 小グリッド20px）でCAD設計図を再現

## カードスタイル
- ボーダー幅: `1px`（細線・精密感）
- 角丸: `4px`（わずかに丸め）
- ホバー: `translateY(-2px)` + ブルーシャドウ
