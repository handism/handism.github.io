# AWSアーキテクチャパターン (データソース)

このディレクトリは、技術ブログの「AWS Patterns」ギャラリー（`/aws-patterns`）のデータソースです。
CloudFormation テンプレート（IaC）やアーキテクチャ図（Draw.io）を管理しています。

## ディレクトリ構成

- `iac/`: 各アーキテクチャのCloudFormationテンプレート（`.yaml`）が格納されています。
- `draw.io/`: アーキテクチャ図の元ファイル（`.drawio`）が格納されています。
- `img/`: エクスポートされたアーキテクチャ図の画像（`.drawio.svg`）が格納されています。
- `gallery-meta.json`: ギャラリーに掲載する全パターンのメタデータ（タイトル、概要、カテゴリ、使用サービス、ファイル名マッピング）を一括管理しています。

## 利用可能なアーキテクチャパターン

`gallery-meta.json` に定義されている各パターンは、ビルド時または開発サーバー起動時に、自動的に `public/aws-patterns/` に複製・同期され、ブログページ上でプレビューやダウンロードができるようになります。

## 注意事項

- アーキテクチャパターンを追加・更新する場合は、`gallery-meta.json` の設定オブジェクトを更新し、対応する `iac/` 下の YAML ファイルや `img/` 下の SVG ファイルを配置してください。
- 元の `.drawio` ファイルは `draw.io/` ディレクトリ配下に保存してあります。今後編集する際にご利用ください。
