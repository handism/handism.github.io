## ショートカット

* Windowsキー + 左右キーでウィンドウを整列させる
- Win + Shift + S：範囲スクショ
- Shift + 右クリック：パスのコピー
- Ctrl + tab ：ブラウザなどのタブ切り替え
- Shift + Win + →←：別モニタにウィンドウ移動
- Win + E：デスクトップ表示
- Win + →←：ウィンドウ整列
